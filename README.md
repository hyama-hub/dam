# 埼玉県のダム 貯水状況ビューア

埼玉県「川の防災情報」管内のダム（下久保・浦山・滝沢・二瀬・有間・合角ほか計9）の
貯水状況をリアルタイムで可視化する非公式のWebアプリです。ダムをプルダウンで切り替えられます。

## 表示内容

- 貯水率（有効貯水容量比）・貯水量・貯水位・流入量・放流量
- 各種**基準水位**（洪水時最高水位・常時満水位・夏季制限水位・最低水位など）を重ねた水位ゲージ
- 過去24時間の貯水量／流入・放流量の推移グラフ

## 起動（ローカル開発）

```sh
node server.js
# → http://localhost:3000
```

依存パッケージなし（Node 18+ の組み込み fetch を使用。チャートは Chart.js を CDN から読み込み）。

## デプロイ（Vercel）

静的ホスティングでは `server.js` が動かないため、`/api/*` は Vercel Serverless Function として動かします。

- `public/` … 静的フロント（Vercelの出力ディレクトリ）
- `api/dam.js` … `GET /api/dam?ccd=XXXX` 指定ダムのライブ値
- `api/dams.js` … `GET /api/dams` ダム一覧（選択用）
- `lib/dam.js` … データ取得ロジック（server.js と api/* で共用）

## データソース

[埼玉県 川の防災情報](https://suibo-river.pref.saitama.lg.jp/) が配信するCSVをプロキシ（CORS回避）して取得しています。

- `chitenconfig/DamList.csv` — ダム一覧と基準水位・容量（静的設定）
- `hyoujidata/dam_detail_{ccd}.csv` — 最新値（観測時刻・放流量・流入量・貯水位・貯水量）
- `hyoujidata/dam_detail_{ccd}_tbl24.csv` — 過去24時間の1時間毎データ

`ccd` は `DamList.csv` に存在するもののみ許可（ホワイトリスト）。観測は約10分毎更新、サーバー側で5分キャッシュ。

履歴は埼玉県ソースの上限である24時間まで。数日〜数ヶ月の長期履歴を出すには別ソースが必要です。

貯水率は「貯水量 ÷ 有効貯水容量」の独自計算値で、公式発表の貯水率（利水容量基準）とは異なる場合があります。

## 注意

非公式ツールです。防災判断には必ず公式情報を参照してください。
