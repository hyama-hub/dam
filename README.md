# 下久保ダム 貯水状況ビューア

下久保ダム（利根川水系神流川）の貯水量をリアルタイムで可視化する非公式のWebアプリです。

## 起動

```sh
node server.js
# → http://localhost:3000
```

依存パッケージなし（Node 18+ の組み込み fetch を使用。チャートは Chart.js を CDN から読み込み）。

## データソース

[埼玉県 川の防災情報](https://suibo-river.pref.saitama.lg.jp/damDetail.html?ccd=12331400006) が配信するCSVを `server.js` がプロキシ（CORS回避）して取得しています。

- `hyoujidata/dam_detail_12331400006.csv` — 最新値（観測時刻・放流量・流入量・貯水位・貯水量・容量）
- `hyoujidata/dam_detail_12331400006_tbl24.csv` — 過去24時間の1時間毎データ

観測は約10分毎更新。サーバー側で5分キャッシュしています。

貯水率は「貯水量 ÷ 有効貯水容量（120,000千m³）」の独自計算値で、公式発表の貯水率（利水容量基準）とは異なる場合があります。

## 注意

非公式ツールです。防災判断には必ず公式情報を参照してください。
